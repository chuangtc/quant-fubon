# quant-fubon

Python and TypeScript version of interaction with Fubon TradeAPI

## Pre-Required

- nvm
- node v22

- python3>=3.10

## Install TypeScript related

```bash
npm install
```

## Use npm run DEV src/index.ts

```bash
npm run dev
```

## Use node run DEV src/index.js

```bash
node src/index.js
```
